# Custom keywords

> Keywords can be seen as properties of a JSON object in a schema.

Adding support for custom keywords is achieved by providing a custom [EvaluatorFactory](https://javadoc.io/doc/dev.harrel/json-schema/latest/dev/harrel/jsonschema/EvaluatorFactory.html) implementation.
For simple use cases, consider using a simple [builder](#evaluator-factory-builder).
